# Suraj's home page

This is a gitrepo to my personal [website](https://freesuraj.github.io). It uses Jekyll to run the website. 

## Running locally

```bash

gem install bundler
bundle install
bundle exec jekyll serve
```

After doing this the server will run at: http://127.0.0.1:4000
